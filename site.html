Kodlama

Metin İşaretleme Dilidir.

HTML (HyperText Markup Language), web sayfalarını oluşturmak için kullanılan standart metin işaretleme dilidir. Bir programlama dili değil, bir yapının nasıl görüneceğini belirleyen bir "iskelet" sistemidir.

HTML'in Rolü (Web'in Üç Atlısı) Bir web sitesini bir binaya benzetirsek: HTML: Binanın kolonları ve duvarlarıdır (Yapı). CSS: Binanın boyası, dekorasyonu ve dış görünüşüdür (Tasarım). JavaScript: Binadaki asansör, ışık sistemi ve otomatlardır (Fonksiyonellik).

CSS

CSS Nasıl Çalışır? (Sözdizimi)

CSS (Cascading Style Sheets), web sayfalarının görsel stilini belirleyen dildir. HTML ile oluşturduğun iskeleti renklendirmek, hizalamak ve güzelleştirmek için kullanılır.

CSS, HTML elementlerini seçer ve onlara özellikler atar. Yapısı şu şekildedir: Seçici (Selector): Hangi HTML öğesini değiştireceğiz? (Örn: h1, p, .buton) Özellik (Property): Neyi değiştireceğiz? (Örn: color, font-size) Değer (Value): Yeni hali ne olacak? (Örn: red, 20px) Python ve AWS altyapısı kullanılmaktadır.

Bir Kez Yaz, Her Yerde Çalıştır" (WORA)

Java'nın en büyük devrimi taşınabilirliğidir. Yazdığın bir Java kodu; Windows, Mac, Linux veya bir buzdolabı ekranında bile aynı şekilde çalışabilir. Bunu sağlayan şey JVM (Java Virtual Machine - Java Sanal Makinesi) sistemidir..

Nesne Yönelimli Programlama (OOP) Java, dünyayı "nesneler" (objects) üzerinden modeller. Örneğin, bir "Araba" sınıfı (Class) oluşturursun; bu arabanın rengi, hızı ve markası onun özellikleri olur. Bu yaklaşım, büyük projelerin parçalara bölünerek çok daha kolay yönetilmesini sağlar.
